# 제 19회 Embedded S/W 경진대회

## 개요



본 레포지토리는 2021년도에 개최된 제 19회 임베디드 S/W 경진대회 - WebOS(LG전자) 대회 본선에서 사용할 소스코드를 저장하는 공간입니다.

Enact를 사용하여 욕조 자동제어를 구현

## 참여자 정보



- **소속 : 한국폴리텍대학 서울강서캠퍼스 데이터분석과**

팀명 : 립싱큐

팀장 : 양준호

팀원 : 정성길, 김하윤, 박건웅

## 개발 환경



- Raspberry Pi
    - WebOS

- Arduino
    - 아두이노 우노 Uno R3 SMD
    - 브레드보드 400홀
    - 모터 드라이버 2A L298N (워터펌프 제어)
    - 아두이노 워터펌프 모터
